# Agri RS AI Portfolio

This repository contains a set of learning-focused remote sensing projects built using Sentinel-2 data and GIS workflows.

These projects are not production systems or publications. They are designed for learning, experimentation, and building a strong technical portfolio in agricultural remote sensing and geospatial data analysis.

---

## Current Projects

### Project 1 — NDVI Classification Stability
This project examines how different NDVI preprocessing pipelines (especially cloud masking strategies) influence field-level agricultural risk classification.

It focuses on measuring how much classification results change when the same satellite data is processed in different realistic ways.

---

### Project 2 — NDVI Scale Sensitivity in Rice Fields
This project studies how spatial resolution and field geometry affect the reliability of NDVI measurements.

It focuses on scale effects such as:
- field size
- edge contamination
- resolution mismatch

and how these factors influence NDVI stability in smallholder agricultural systems.
